# Site do Minicurso de Bioinformática – Curso de Inverno 2026

Este repositório contém a página oficial do **Minicurso de Bioinformática Básica e Aplicada à Genômica e Transcriptômica**, oferecido durante o VIII Curso de Inverno do PPGBB no Instituto Carlos Chagas (ICC/Fiocruz).

## Acesse o site

[https://icc-biogc.github.io/2026-Curso_Inverno/](https://icc-biogc.github.io/2026-Curso_Inverno/)

## Estrutura de arquivos

```
2026-Curso_Inverno/
│
├── index.html                          # Página principal do minicurso
├── Ementa_Bioinformatica_2026.docx.pdf # Ementa completa do curso
├── README.md                           # Este arquivo
│
├── css/
│   └── style.css                       # Estilos visuais (paleta institucional)
│
├── materiais/                          # Pasta com os materiais das aulas
│   ├── aula1_bioinfo.pdf               # Introdução ao Unix
│   ├── aula2_bioinfo.html              # Pipeline de RNA-Seq
│   ├── genomica_bacteriana.pdf         # Montagem e anotação de genomas
│   └── variantes_humanas.pdf           # Chamada de variantes em WES/WGS
│
└── img/                                # (opcional) Imagens e logos
    └── (arquivos de imagem)
```

## Como contribuir ou atualizar

1. Faça um clone do repositório
2. Edite os arquivos necessários (HTML, CSS ou adicione PDFs na pasta `materiais/`)
3. Faça commit e push para a branch `main`
4. O GitHub Pages atualizará o site automaticamente
